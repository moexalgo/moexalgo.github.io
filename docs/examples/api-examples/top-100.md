Большинство из этих запросов доступно через [подписку ALGOPACK](https://data.moex.com/products/algopack?utm_source=algopack_docs&utm_medium=internal_link&utm_campaign=algopack_docs_v2&utm_content=api_example_top100)

После оформления подписки в личном кабинете получаете APIKEY и с ним делаете запросы к данным. Пример запроса:

```python
import requests 

url = "https://apim.moex.com/iss/engines/futures/markets/forts/boardgroups/45/securities/SiU5/candles.json"
headers = { 'Authorization': 'Bearer APIKEY', } 

response = requests.request("GET", url, headers=headers) 
print(response.text)
```
