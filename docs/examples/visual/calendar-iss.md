# ALGOPACK VISUAL: CALENDAR

Интерфейс **торгового календаря** показывает торговые и неторговые дни, дополнительные торговые сессии выходного дня (ДСВД) и оперативные сведения по выбранной дате для фондового и срочного рынков. Календарные данные доступны в **машиночитаемом виде** через ALGOPACK

![Торговый календарь в ALGOPACK](/img/calendar.png)

Визуализация доступна на странице [MOEX](https://moex.com).

:::note

Расширенное описание таблиц, полей и эндпоинтов по рынкам — в разделе [API](/docs/api/) и [Машиночитаемый календарь ИСС](/docs/description/calendar-iss). Руководство к ИСС с метаданными и примерами для разработчиков: [moex.com/a2193](https://www.moex.com/a2193).

:::
