# ALGOPACK VISUAL: HI2

Визуализация индексов рыночной концентрации (HI2) для анализа распределения торговой активности между участниками рынка. Индекс рыночной концентрации (Herfindahl-Hirschman Index) показывает насколько торговля в акциях сосредоточена у крупных трейдеров или распределена между множеством небольших трейдеров.

Чем больше значение индекса, тем больше влияние крупных трейдеров на инструмент. [Подробнее](/docs/modules/hi2).
