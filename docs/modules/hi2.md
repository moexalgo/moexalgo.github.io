# Индекс рыночной концентрации (HI2)

### Что это?
Индекс Херфиндаля-Хиршмана (HI2) — это ключевой показатель для оценки концентрации рынка и уровня конкуренции среди участников. 
HI2 определяется как сумма квадратов рыночных долей всех участников 

В экономике HI2 используется для анализа конкурентной среды на рынках, 
выявления доминирующих участников и оценки возможных рисков, связанных с монополизацией.


### Формула

HI2 = сумма квадратов рыночных долей всех участников (s_i)^2, где s_i — рыночная доля i-го участника в процентах


### Пример расчета

Рассмотрим более детально пример с тремя компаниями:

1. Компания A имеет рыночную долю 50%. 
2. Компания B имеет рыночную долю 30%.
3. Компания C имеет рыночную долю 20%.

Рассчитаем HI2:

HI2 = 50² + 30² + 20² = 2500 + 900 + 400 = 3800

### Что означает высокий и низкий HI2?

![image](/img/hi2_range.png)

- **Низкий HI2** (< 1500): Указывает на высокую конкуренцию на рынке. Большое количество участников имеет равные рыночные доли.
- **Средний HI2** (1500 - 2500): Указывает на умеренную концентрацию. Несколько участников могут иметь более значительные доли рынка.
- **Высокий HI2** (> 2500): Указывает на высокую концентрацию рынка. Несколько участников доминируют на рынке.

:::note
    Индексы HI2 рассчитываются по акциям первого и второго котировального списка, фьючерсным контрактам и валюте
	
	Если количество трейдеров в инструменте менее определенного порога, то за этот день по этому инструменту рассчеты не производятся
	
	Наиболее ликвидные инструменты торгуют десятки и сотни тысяч трейдеров за торговый день
:::

### Множество метрик HI2

Кроме объема торгов, HI2 рассчитывается в разрезе покупок и продаж, агрессивных и пассивных сделок:

![image](/img/hi2_tree.png)

Такое разбиение позволяет лучше понять рыночную концентрацию. Если один индикатор показывает низкое значение, 
другой может показать высокое. Пример, наложение цены на индикаторы HI2:

![image](/img/hi2_sber.png)

* **hhi_volume** - по объему торгов
* **hhi_buy** - по объему покупок
* **hhi_sell** - по объему продаж
* **hhi_netflow_buy** - по нетто-объему (покупки преобладают над продажами)
* **hhi_netflow_sell** - по нетто-объему (продажи преобладают над покупками)
* **hhi_passive** - по мейкерским объемам
* **hhi_active** - по тейкерским объемам
* **hhi_passive_buy** - по мейкерским объемам (покупки)
* **hhi_active_buy** - по тейкерским объемам (продажи)
* **hhi_passive_sell** - по мейкерским объемам (продажи)
* **hhi_active_sell** - по тейкерским объемам (покупки)

> *volume = buy + sell = passive + agressive*

### Для кого?
* Крупных трейдеров и аналитиков
* Брокеров
* Маркет-мейкеров 
* Алготрейдеров (систематическая торговля)


### Преимущества

Уникальная аналитика MOEX о концентрации рынка дает участникам дополнительные возможности при принятии торгового решения:

* при покупке (продаже) актива в портфель
* при установлении максимальных лимитов на позиции
* при настройке агрессивности входа и выхода из позиций

### Данные
* Показатели рассчитываются ежедневно и предоставляются в конце торгового дня
* Для каждого инструмента рассчитывается 11 показателей HI2
* Инструменты: Акции, Фьючерсы, Валюта
* Исторические данные с 2020 года
* Данные доступны через REST API
