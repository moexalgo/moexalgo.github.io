# Futures Open Interest (FUTOI)
Открытые позиции по фьючерсным контрактам в разрезе физических и юридических лиц 

Продукт позволяет в течение дня получать статистику по открытым позициям физических и 
юридических лиц с пятиминутными интервалами по наиболее ликвидным фьючерсным контрактам: 
BR, Si, RI, GD…

* Это уникальные данные о динамике рынка и настроениях инвесторов
* Продукт дает повышенную прозрачность, разбивая сделки по типам участников рынка и направлению buy/sell
* Продукт может быть использован в качестве дополнительного фактора при построении системных стратегий

Данные доступны через API и в графическом интерфейсе на [странице](https://data.moex.com/products/algopack/visual?tab=futoi&code=Si)
