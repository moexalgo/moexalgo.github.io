Авторизация и получение APIKEY
******************************

Для неавторизованных пользователей данные предоставляются в
ограниченном наборе и задержкой по времени. Чтобы получить доступ к
полному набору данных, надо зарегистрироваться на data.moex.com,
оформить подписку на странице ALGOPACK и получить APIKEY.

Оптимальный способ передать APIKEY в код используя переменную среды:

   import os
   from moexalgo import session, Market
   session.TOKEN = os.environ['APIKEY']

   eq = Market('EQ')
   eq.tickers()
