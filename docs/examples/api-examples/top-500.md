Большинство из этих запросов доступно через [подписку ALGOPACK](https://data.moex.com/products/algopack)

После оформления подписки в личном кабинете получаете APIKEY и с ним делаете запросы к данным. Пример запроса:

```python
import requests 

url = "https://apim.moex.com/iss/engines/futures/markets/forts/boardgroups/45/securities/SiU5/candles.json"
headers = { 'Authorization': 'Bearer APIKEY', } 

response = requests.request("GET", url, headers=headers) 
print(response.text)
```
